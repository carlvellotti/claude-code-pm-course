# Claude Code PM 课程

这是一门互动课程，旨在教导产品经理如何在日常工作中有效地使用 Claude Code。

## 课程结构

### 模块 0：入门指南
- **0.0 简介** - 课程概览和学习内容
- **0.1 安装** - 安装 Claude Code 和前提条件
- **0.2 启动与克隆** - 启动 Claude Code 并克隆本课程

### 模块 1：Claude Code 基础
- **1.1 欢迎** - 介绍 TaskFlow 和课程项目
- **1.2 可视化文件** - 设置您的可视化工作区（Nimbalyst、Obsidian 或 VS Code）
- **1.3 初始任务** - 处理会议记录、分析研究、处理图像
- **1.4 智能体 (Agents)** - 使用并行智能体处理复杂工作流
- **1.5 自定义子智能体** - 创建专门的 AI 角色进行审查
- **1.6 项目记忆** - 使用 CLAUDE.zh.md 获取上下文和说明
- **1.7 Claude Code 导航** - 掌握文件操作和搜索

### 模块 2：高级 PM 场景
- **2.1 撰写 PRD** - 与 AI 合作创建产品需求文档
- **2.2 分析数据** - 数据驱动的产品决策
- **2.3 产品策略** - 战略规划和竞争分析

## 如何使用本课程

### 互动轨道（推荐）
1. 克隆此仓库
2. 导航到课程材料目录：`cd course-materials`
3. 启动 Claude Code：`claude`
4. 运行 `/start-1-1` 开始第一课
5. 跟随指导进行动手练习

### 参考轨道
每个模块都有一个 `REFERENCE_GUIDE.md`，可以单独阅读以供快速参考。

## ⚠️ 重要提示：Claude Code 说明

**当您首次在 Claude Code 中打开此仓库时：**
- ❌ **请勿安装 npm 依赖项** 或运行 `npm install`
- ❌ **请勿构建网站** 或运行 `npm run build`
- ❌ **请勿进行任何设置更改**，除非被要求

**相反：**
- 📖 按照指导阅读课程内容
- 🎯 等待每个模块中的明确指示
- 💬 询问用户下一步想做什么

本课程是独立的 - 我们只在需要时处理设置。您的工作是引导用户进行交互式学习体验。

## 您将学到什么

- **文件操作** - 读取、搜索、组织和管理 PM 文档
- **@-提及** - 为 Claude 提供完整的上下文以获得更好的输出
- **智能体** - 运行并行工作流并获得多角度反馈
- **自定义子智能体** - 创建专门的审阅者（工程师、高管、UX）
- **项目记忆** - 使用 CLAUDE.zh.md 自定义 Claude 的行为
- **真实 PM 工作流** - 撰写 PRD、分析数据、制定策略

## 前提条件

- 对产品管理有基本的了解
- 愿意学习命令行基础知识
- Mac、Windows 或 Linux 电脑

## 关于本课程

本课程教导 PM 如何将 Claude Code 用作思维伙伴，而不仅仅是自动化工具。您将学习如何：
- 在不牺牲质量的情况下更快地工作
- 即时获得多角度反馈
- 高效处理研究和数据
- 在 AI 协助下撰写更好的文档

**完成时间：** 完整互动轨道约 4-6 小时
