## 0.0: 介绍

## 📖 概览

Claude Code 是一个直接与你计算机上的文件和文件夹协作的 AI 助手。与传统的需要你手动复制粘贴上下文的聊天机器人不同，Claude Code 直接通过自然对话阅读你的文档、分析数据并创建资料。

**关键要点:** Claude Code 将产品经理从手动执行者转变为协调者。你专注于战略决策，而 Claude 处理文档合成、数据分析和工作流自动化。


## 🎯 你将构建什么

本课程教你使用 Claude Code 构建一个产品经理生产力系统。

**核心能力:**
1. **文件操作** - 使用 @ 符号读取、写入和引用文件
2. **视觉分析** - 粘贴截图并分析模型、图表、数据
3. **并行智能体** - 同时运行多个 AI 实例进行批处理
4. **自定义子智能体** - 拥有个性的永久专业团队成员
5. **项目记忆** - 通过 CLAUDE.zh.md 维护关于产品和团队的持久上下文


## 📋 要求

- Claude Pro 或 Max 订阅 ($20/月)
- Mac, Windows, 或 Linux 计算机
- 15 分钟安装时间

**时间投入:**
- 等级 1 (基础): 2-3 小时
- 等级 2 (从业者): 4-5 小时
- 等级 3 (大师): 3-4 小时 [即将推出]

总计: 10-12 小时成为高级用户。每周节省 5-10 小时。


## 💼 真实产品经理用例

### 战略工作
- **PRD 撰写** - 将研究、访谈和竞争分析综合成连贯的文档
- **路线图规划** - 分析市场数据和利益相关者输入
- **客户反馈分析** - 在几分钟内处理数百条反馈
- **竞争情报** - 生成 SWOT 分析和比较矩阵

### 执行与组织
- **工单生成** - 将功能规格转换为用户故事和 Jira 工单
- **多格式转换** - 将 PRD 转换为单页文档、演示文稿、发布说明
- **文件组织** - 构建研究文件、会议记录、截图
- **数据编辑** - 无需打开 Excel 即可操作电子表格和 CSV

### 沟通
- **利益相关者更新** - 起草电子邮件、Slack 消息、Notion 页面
- **发布说明** - 将工程变更日志转换为客户友好的语言
- **演示叙事** - 从战略文档构建幻灯片
- **知识库** - 跨会话维护持久的产品知识

### 研究与分析
- **SEO 审计** - 分析营销页面和内容
- **竞争对手研究** - 创建比较表和功能矩阵
- **洞察合成** - 结合用户访谈、调查、支持工单、应用评论
- **网络研究** - 自动化竞争情报收集


## 🔄 Claude Code vs 传统聊天机器人

| 特性 | 传统聊天机器人 | Claude Code |
|---------|---------------------|-------------|
| **文件访问** | 无 | 读/写你计算机上的任何文件 |
| **记忆** | 会话后消失 | 永久项目记忆 (CLAUDE.zh.md) |
| **执行** | 建议，但你来做 | 实际执行命令并创建文件 |
| **多文件** | 一次性响应 | 同时跨数十个文档工作 |
| **智能体** | 单个实例 | 并行运行多个智能体 |
| **集成** | 有限 | 通过 MCP 连接 Notion, Linear, Slack, GitHub |
| **版本控制** | 无历史 | Git 集成、检查点、回退 |
| **自动化** | 手动复制粘贴 | 通过自定义命令实现真正的自动化 |

**区别:** 持久性和执行力。Claude Code 不仅仅给出建议——它和你一起工作。


## 📊 真实案例: Monday.com 产品经理

Monday.com 的一位产品经理使用 Claude Code 收集并分析了关于他产品的 34,000 条 Reddit 评论。Claude 识别出了直接影响路线图的模式、痛点和优先主题。

**影响:** 数周的手动工作 → 几小时。这不关乎取代产品经理的判断——而是处理繁重的工作，以便你专注于战略思考。


## 🧠 思维转变

> "Claude Code 代表着开发者变成极其强大系统的协调者，而不仅仅是命令打字员。" - Peter Steinberger, PDF Expert 创作者

这也适用于产品经理。你不是在学习编码——你是在学习协调。

**转变:**
- 从: "我需要学习这个命令"
- 到: "我需要这个结果"

Claude Code 在终端中运行，这可能会让你感到陌生。但你不会学习终端命令——你将进行对话。到模块 0 结束时，终端只是你与 Claude 交谈的地方。
