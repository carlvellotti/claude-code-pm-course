# 翻译任务计划：course-materials Python 脚本本地化

## 1. 任务目标
将 `course-materials` 目录下的 4 个核心 Python 脚本进行中文本地化，包括代码注释、文档字符串 (Docstrings) 和用户可见的打印输出 (Print statements)。

## 2. 翻译范围
### 目标文件
1. `course-materials/lesson-modules/3-nano-banana/image_gen.py`
2. `course-materials/lesson-modules/3-nano-banana/style_extract.py`
3. `course-materials/lesson-modules/3-nano-banana/3.1-intro-to-image-gen/3.1.4-style-database/get_style.py`
4. `course-materials/lesson-modules/2-advanced-pm-work/2.2-analyze-data/generate_data.py`

### 翻译规则
1. **Docstrings (`"""..."""`)**: 翻译为简洁专业的中文，保持原有格式参数描述。
2. **行内注释 (`# ...`)**: 翻译为中文，解释代码逻辑。
3. **打印输出 (`print("...")`)**: 翻译提示性文本，便于学生理解交互。
4. **功能性字符串 (保留英文)**:
   - `STYLE_EXTRACTION_PROMPT` (在 `style_extract.py` 中): **绝对保留英文**，这是发给模型的指令。
   - 字典键值 (Key/Value)、API 参数、环境变量名、文件路径。
   - 错误信息 (`raise ValueError("...")`)：翻译错误描述，保留关键变量名。
