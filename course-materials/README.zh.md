# Claude Code for Product Managers - 课程材料

欢迎来到 Claude Code 产品经理互动课程！

## 快速开始

**你来对地方了！** 这个目录包含完成课程所需的一切。

### 包含内容

- **`lesson-modules/`** - Claude 授课的互动模块（模块 0-2）
- **`company-context/`** - 练习用的 TaskFlow 公司参考资料
- **`.claude/`** - 支撑课程体验的斜杠命令

### 如何开始课程

**1. 确认当前目录：**
```bash
pwd
# 应该显示：.../course-materials
```

**2. 启动 Claude Code：**
```bash
claude
```

**3. 开始第一个模块：**

Claude Code 运行后，输入：
```
/start-1-1
```

这将启动模块 1.1 - 欢迎来到 Claude Code！

### 课程结构

**模块 0：入门**（前置条件）
- 在运行本课程前，应已完成模块 0（安装与设置）
- 若尚未安装 Claude Code，请访问：https://ccforpms.com/getting-started/installation

**模块 1：基础**（从 `/start-1-1` 开始）
- 1.1：欢迎
- 1.2：可视化文件
- 1.3：第一次任务
- 1.4：Agents（并行处理）
- 1.5：自定义子 Agent
- 1.6：项目记忆
- 1.7：导航

**模块 2：高级 PM 工作**
- 2.1：撰写 PRD
- 2.2：数据分析
- 2.3：产品策略

### 斜杠命令

在课程中你会使用如下命令：
- `/start-1-1` - 开始模块 1.1
- `/start-2-1` - 开始模块 2.1
- 等等

这些命令定义在 `.claude/` 目录中，引导你完成每个模块的互动流程。
